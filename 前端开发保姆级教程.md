# EcoWise 宿舍助理 - 前端开发保姆级教程(从零开始)

> 这份教程面向零基础同学,从装环境到跑通小程序,每一步都有详细说明。

---

## 一、先理解你要做什么

### 你要开发的是"涂鸦智能小程序"

- **运行在哪**:涂鸦智能生活 App 里(不是微信)
- **用什么写**:Ray 框架(类似 React)+ JavaScript
- **需要什么**:涂鸦账号 + 涂鸦小程序 IDE + 你后端给的 Agent ID

### 小程序要实现的功能(L0)

1. **首页**:展示插座实时数据(功率/电压/电量)
2. **AI对话页**:集成智能体,用户能跟 AI 聊宿舍问题
3. **电费分摊页**:展示每人电费
4. **违规检测页**:展示当前用电状态(正常/预警/违规)
5. **历史用电页**:今日+本月用电趋势图

---

## 二、环境准备(一次性,做完就不用再做)

### Step 1:安装 Node.js

1. 去 https://nodejs.org/ 下载 LTS 版本(选 18.20.8 或更高)
2. 双击安装包,一路下一步,全部默认
3. 验证:打开 PowerShell,输入:
   ```bash
   node -v
   ```
   应该显示类似 `v18.20.8`

### Step 2:安装 Yarn

在 PowerShell 里运行:
```bash
npm install -g yarn
```
验证:
```bash
yarn -v
```
应该显示版本号,比如 `1.22.19`

### Step 3:下载涂鸦小程序 IDE

1. 去涂鸦官方下载页:https://developer.tuya.com/cn/miniapp/devtools/tools/download
2. 下载 Windows 版本(选稳定版)
3. 双击安装,一路下一步
4. 安装完成后,打开 IDE,用你的**涂鸦开发者平台账号**登录

## 三、在涂鸦平台创建小程序(在浏览器里操作)

### Step 1:登录小程序开发者平台

1. 浏览器打开:https://platform.tuya.com/miniapp/
2. 用你的涂鸦账号登录
3. 应该能看到「小程序管理」页面

### Step 2:创建新小程序

1. 点「新建」或「创建小程序」
2. 填写:
   - 小程序名称:`EcoWise 宿舍助理`
   - 小程序描述:`寝室 AI 助理 - 用电安全/作息健康/电费分摊`
   - 小程序类型:**智能小程序**(选这个,不是面板小程序)
     - 智能小程序:不依赖具体设备,可以自己写页面逻辑(适合你的场景)
     - 面板小程序:专门做设备控制面板(不适合你)
3. 点「确定」创建

### Step 3:记下小程序 ID

创建完成后,在小程序列表里能看到你的小程序,记下:
- **小程序 ID**(miniProgramId):一串字符串,后面要用

### Step 4:配置小程序能力(Kit)

进入小程序详情,找到「Kit 管理」或「能力配置」,确保这些 Kit 已启用:
- BaseKit(基础组件)
- BizKit(业务组件)
- DeviceKit(设备能力)
- HomeKit(家庭能力)
- MiniKit(小程序基础)
- **AIStreamKit**(AI 对话能力,关键!)

如果 AIStreamKit 没启用,先启用它。

## 四、在 IDE 里创建项目代码

### Step 1:打开涂鸦小程序 IDE

1. 打开涂鸦小程序 IDE
2. 用涂鸦账号登录
3. 登录后应该看到「项目管理」页面

### Step 2:新建项目

1. 点「新建」按钮
2. 填写:
   - **项目路径**:选一个空文件夹,比如 `D:\VS\code\EcoWiseMiniApp`
   - **项目名称**:`EcoWise宿舍助理`
   - **所属开发者平台小程序**:选你刚才创建的那个小程序
3. 点「下一步」
4. 在「选择模板」页面,选 **"App 通用模板"** 或 **"Ray 应用模板"**(不要选面板模板)
5. 点「创建」

### Step 3:等待自动安装依赖

IDE 会自动跑 `yarn install`,等几分钟,看到控制台出现 `compiled successfully` 就成功了。

### Step 4:扫码登录授权

1. 在 IDE 顶部点账户图标,出现二维码
2. 用手机打开**涂鸦智能生活 App**(没装就去应用商店搜"Smart Life"或"智能生活"下载)
3. 用 App 扫描 IDE 里的二维码,确认登录
4. 登录后,IDE 会获取到你的家庭和设备信息

### Step 2:修改首页代码

打开 `src/pages/home/index.tsx`,替换成:

```typescript
import React, { useState, useEffect } from 'react';
import { View, Text, Button, ScrollView } from '@ray-js/components';
import { devices } from '@ray-js/panel-sdk';

export default function Home() {
  const [deviceData, setDeviceData] = useState({
    power: 0,
    voltage: 0,
    current: 0,
    energy: 0,
    online: false,
  });

  // 获取设备数据
  const fetchDeviceData = async () => {
    try {
      // 这里的 deviceId 换成你后端同学给的设备 ID
      const deviceId = '6cc488dba84ddfba58ypx2';
      const dpState = await devices.getDeviceDps(deviceId);
      console.log('DP 数据:', dpState);

      setDeviceData({
        power: (dpState.cur_power || 0) * 0.1,        // W
        voltage: (dpState.cur_voltage || 0) * 0.1,    // V
        current: dpState.cur_current || 0,             // mA
        energy: (dpState.add_ele || 0) * 0.001,        // kWh
        online: true,
      });
    } catch (err) {
      console.error('获取设备数据失败:', err);
    }
  };

  useEffect(() => {
    fetchDeviceData();
    // 每 5 秒刷新一次
    const timer = setInterval(fetchDeviceData, 5000);
    return () => clearInterval(timer);
  }, []);

  return (
    <ScrollView style={{ padding: 20, backgroundColor: '#f5f5f5' }}>
      <Text style={{ fontSize: 24, fontWeight: 'bold', marginBottom: 20 }}>
        EcoWise 宿舍助理
      </Text>

      {/* 实时数据卡片 */}
      <View style={{
        backgroundColor: '#fff',
        padding: 20,
        borderRadius: 12,
        marginBottom: 16,
      }}>
        <Text style={{ fontSize: 16, color: '#666', marginBottom: 8 }}>
          插座1(同学A-电脑桌)
        </Text>
        <Text style={{ fontSize: 14, color: deviceData.online ? '#52c41a' : '#999' }}>
          {deviceData.online ? '● 在线' : '● 离线'}
        </Text>
      </View>

      {/* 功率 */}
      <View style={cardStyle}>
        <Text style={labelStyle}>当前功率</Text>
        <Text style={valueStyle}>{deviceData.power.toFixed(1)} W</Text>
      </View>

      {/* 电压 */}
      <View style={cardStyle}>
        <Text style={labelStyle}>当前电压</Text>
        <Text style={valueStyle}>{deviceData.voltage.toFixed(1)} V</Text>
      </View>

      {/* 电流 */}
      <View style={cardStyle}>
        <Text style={labelStyle}>当前电流</Text>
        <Text style={valueStyle}>{deviceData.current} mA</Text>
      </View>

      {/* 累计电量 */}
      <View style={cardStyle}>
        <Text style={labelStyle}>累计电量</Text>
        <Text style={valueStyle}>{deviceData.energy.toFixed(3)} 度</Text>
      </View>

      <Button
        title="刷新数据"
        onClick={fetchDeviceData}
        style={{ marginTop: 20 }}
      />
    </ScrollView>
  );
}

const cardStyle = {
  backgroundColor: '#fff',
  padding: 16,
  borderRadius: 12,
  marginBottom: 12,
};

const labelStyle = {
  fontSize: 14,
  color: '#999',
  marginBottom: 4,
};

const valueStyle = {
  fontSize: 20,
  fontWeight: 'bold',
  color: '#333',
};
```

### Step 3:编译运行

## 七、添加 AI 对话页面

### Step 1:安装 AI 对话 SDK

在 IDE 顶部菜单找「终端」→「新终端」,运行:

```bash
yarn add @ray-js/t-agent@^0.2.0 @ray-js/t-agent-plugin-aistream@^0.2.0 @ray-js/t-agent-ui-ray@^0.2.0
```

三个包版本必须一致。

### Step 2:确认 package.json 里 Kit 配置

打开 `package.json`,确认 dependencies 里有:

```json
"dependencies": {
  "BaseKit": "3.12.0",
  "BizKit": "4.10.0",
  "DeviceKit": "4.6.1",
  "HomeKit": "3.4.0",
  "MiniKit": "3.12.1",
  "AIStreamKit": "1.0.0"
}
```

关键:**AIStreamKit: "1.0.0" 必须有**。

### Step 3:创建 AI 对话页面

新建文件夹 `src/pages/chat/`,在里面新建 `index.tsx`:

```typescript
import React from 'react';
import { View } from '@ray-js/components';
import { createChatAgent, withDebug, withUI } from '@ray-js/t-agent';
import { ChatContainer, MessageInput, MessageList, MessageActionBar } from '@ray-js/t-agent-ui-ray';
import { withAIStream, withBuildIn } from '@ray-js/t-agent-plugin-aistream';

// ⚠️ 把这里换成后端给你的 Agent ID
const AGENT_ID = 'agent_xxxxxx';

const createAgent = () => {
  const agent = createChatAgent(
    withUI(),
    withAIStream({
      agentId: AGENT_ID,
      enableTts: false,
      earlyStart: true,
      tokenOptions: {
        api: 'm.life.ai.token.get',
        version: '1.0',
      },
    }),
    withDebug(),
    withBuildIn()
  );
  return agent;
};

export default function ChatPage() {
  return (
    <View style={{ height: '100vh', backgroundColor: '#fff' }}>
      <ChatContainer createAgent={createAgent}>
        <MessageList />
        <MessageInput />
        <MessageActionBar />
      </ChatContainer>
    </View>
  );
}
```

### Step 4:测试 AI 对话

1. 编译
2. 在模拟器或真机进入 `pages/chat/index` 页面
3. 输入"你好"发送,几秒后应该收到 AI 回复
4. 如果没响应,检查 console 日志

### Step 2:创建历史页面

```typescript
import React, { useState } from 'react';
import { View, Text, ScrollView } from '@ray-js/components';

export default function HistoryPage() {
  const [dailyData] = useState([
    { date: '07-01', kwh: 0.8 },
    { date: '07-02', kwh: 1.2 },
    { date: '07-03', kwh: 0.6 },
    { date: '07-04', kwh: 1.5 },
    { date: '07-05', kwh: 0.9 },
    { date: '07-06', kwh: 1.1 },
    { date: '07-07', kwh: 0.7 },
  ]);

  const maxKwh = Math.max(...dailyData.map(d => d.kwh));

  return (
    <ScrollView style={{ padding: 20 }}>
      <Text style={{ fontSize: 24, fontWeight: 'bold' }}>本月用电趋势</Text>

      <View style={{ marginTop: 20 }}>
        {dailyData.map((item, idx) => (
          <View
            key={idx}
            style={{
              flexDirection: 'row',
              alignItems: 'center',
              marginBottom: 12,
            }}
          >
            <Text style={{ width: 60, fontSize: 12, color: '#666' }}>
              {item.date}
            </Text>
            <View
              style={{
                height: 20,
                backgroundColor: '#1890ff',
                width: `${(item.kwh / maxKwh) * 100}%`,
                borderRadius: 4,
                marginRight: 8,
              }}
            />
            <Text style={{ fontSize: 12 }}>{item.kwh} 度</Text>
          </View>
        ))}
      </View>
    </ScrollView>
  );
}
```

## 十三、常见问题排查

### 问题 1:模拟器白屏

- 看 console 报错
- 常见原因:Kit 版本不对,去「环境配置」→「Kit 管理」选推荐版本

### 问题 2:获取设备数据失败

- 确认 deviceId 对不对
- 确认你登录的涂鸦账号能控制那个设备(设备要在你的家庭里)
- 在智能生活 App 里先把插座添加到你的家庭

### 问题 3:AI 对话没响应

- 确认 Agent ID 填对了
- 确认智能体已经投放到你的小程序对应的 App
- 看 console 里 `m.life.ai.token.get` 接口是否成功

### 问题 4:设备不在我的家庭里

1. 让后端同学在智能生活 App 里,把设备分享给你
2. 或者你自己配网一个设备,用它的 deviceId

## 十四、下一步:接入后端 API(进阶)

上面首页的数据是小程序直接调涂鸦 API 拿的,没用你的 Python 后端。

L1 阶段可以改成:Python 后端用 Flask 封装成 HTTP API,小程序用 fetch 调用。

```typescript
// 小程序里调后端 API 示例
const response = await fetch('http://你的电脑IP:5000/api/devices');
const data = await response.json();
```

但需要:
1. 电脑和手机在同一 WiFi
2. Python 后端用 Flask 起 HTTP 服务
3. 小程序配置允许 http 域名(开发模式)

这个先不做,L0 先用涂鸦原生 API 把界面跑通。
